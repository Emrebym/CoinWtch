# 📈 CoinWtch

A real-time cryptocurrency market watcher that displays live prices, market caps, and price changes for popular digital assets.

## Features

- **Live price tracking** — real-time updates from crypto APIs
- **Market overview** — top coins by market cap at a glance
- **Price change indicators** — 24h percentage changes with color coding
- **Search & filter** — quickly find specific coins
- **Responsive design** — works on desktop and mobile
